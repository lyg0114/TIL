# 로컬환경에서 생성한 git을 github와 연동

1. 디렉토리에 git을 생성해 준다.
    > git init

2. git remote add origin [copied web address] 명령어를 친다.
    > ex) git remote add origin https://github.com/mindplace/test-repo.git

3. github에 push 한다.
    > git push origin master  


 ### 작업하기 전에 밑의 작업을 해주어야 한다.
    * git add .
    * git commit
